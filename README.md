# Warehouse Placement Recommender API - веб-сервис рекомендаций размещения товара

## Название и назначение сервиса

**Warehouse Placement Recommender API** — это учебный веб-сервис, предоставляющий RESTful API для автоматизированного определения оптимальной складской ячейки размещения товара.  
Сервис анализирует характеристики товара и параметры складских ячеек и возвращает наиболее подходящий вариант размещения в формате JSON.

---

## Назначение

Веб-сервис предназначен для демонстрации принципов разработки Web API на языке Python с использованием фреймворка FastAPI.  
Проект моделирует интеллектуальную систему размещения товаров на складе.

---

## Основные функции сервиса

Сервис предоставляет следующие функции:

1. Получение рекомендации по размещению товара  
2. Проверка соответствия габаритов с учетом возможного поворота  
3. Контроль ограничений по весу  
4. Учет заполненности ячеек  
5. Учет параметров IoT  
6. Формирование уникального идентификатора размещения (PlacementID)  

---

## Архитектурные особенности

### Используемые технологии

- Язык программирования: **Python 3.12+**
- Веб-фреймворк: **FastAPI**
- ASGI-сервер: **Uvicorn**
- Валидация данных: **Pydantic**
- Документация API: **Swagger (OpenAPI)**

### Хранение данных

- Данные передаются в HTTP-запросе  
- Хранение осуществляется в оперативной памяти  
- База данных не используется (учебный проект)

---

## Инструкция по развертыванию и запуску

### Требования

- Установленный Python 3.12+  
- Операционная система Windows / Linux / macOS  

### Шаги запуска

1. Клонировать репозиторий или скопировать исходный код
2. Инициализировать и запустить проект:
```bash
    # Создать виртуальное окружение:
    python -m venv .venv
    # Активировать окружение:
    .venv\Scripts\activate
    # Установить зависимости:
    pip install fastapi uvicorn httpx
    # Запустить сервер:
    python main.py
```
